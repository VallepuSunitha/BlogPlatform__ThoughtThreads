# 📝 ThoughtThreads - Django Blog Platform

ThoughtThreads is a full-stack blog platform built using Django. It allows users to create, edit, and delete blog posts, manage authentication, and reset passwords via email.


## 🚀 Features

* 🔐 User Authentication (Login, Logout, Register)
* ✍️ Create, Edit, Delete Blog Posts
* 📄 View Post Details
* 📧 Password Reset via Email
* 🎨 Responsive UI using Bootstrap & Crispy Forms
* 🖼️ Media Upload Support (Profile Images, etc.)



## 🛠️ Tech Stack

* **Backend:** Django
* **Frontend:** HTML, CSS, Bootstrap
* **Database:** SQLite (Development)
* **Forms:** Django Crispy Forms
* **Static Handling:** WhiteNoise
